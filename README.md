## Technologies
- Nest.js
- MongoDB
- Docker

---
## Commands
To start our environment we need to enter at the cosrrect folder ans type the following commands:

- Database (MongoDB)
```
sudo docker compose -f docker-compose-dev.yml up
```
- Nestjs (Backend)
```
npm run start:dev
```
- React.js (Frontend)
```
npm run dev
```
